# 行情牛跑酷隐私政策

生效日期：2026 年 6 月 15 日

行情牛跑酷重视用户隐私。本隐私政策说明扩展如何处理信息。

## 数据收集

行情牛跑酷不收集、不存储、不出售，也不上传任何个人信息、浏览历史、账户信息或行情数据。

## 本地处理

扩展仅在用户打开受支持的同花顺个股行情页面时运行。为了生成跑酷游戏路径，扩展会在用户浏览器本地临时读取当前页面上可见的 Canvas 或 SVG 行情图及页面尺寸。

这些信息只用于在当前页面中运行游戏，不会离开用户设备，也不会被持久化。

## 权限用途

扩展仅请求访问 `https://stockpage.10jqka.com.cn/*`，用于读取当前可见行情图并显示游戏覆盖层。扩展不会访问其他网站。

## 第三方共享

行情牛跑酷不会向任何第三方共享、出售或传输用户数据。

## 远程代码与外部服务

扩展不下载或执行远程代码，不使用分析服务、广告服务或外部数据服务。

## 政策更新

如果本隐私政策发生变化，更新后的内容将发布在本页面，并修改生效日期。

## 联系方式

如对本隐私政策有疑问，请通过 Chrome Web Store 中显示的开发者联系方式联系开发者。
